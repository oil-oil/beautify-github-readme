# README 定制练习

这里保存的是 `beautify-github-readme` 针对真实开源项目做的本地概念稿。它们用于检验同一套方法能否适应不同项目，而不是给项目套一张统一模板。

这些设计目前没有提交给对应项目，也没有在图片里加入 `beautify-github-readme` 的名称、Logo 或反链。若将来联系维护者，应先发一张无品牌预览，由对方决定是否需要源文件或 PR。
